# 💳 Credit Card Approval Prediction

A Machine Learning and Flask-based web application that predicts whether a credit card application is likely to be approved based on applicant details. The system helps financial institutions make faster, more consistent, and data-driven credit approval decisions.



# 📖 Project Overview

Credit card approval is one of the most important processes in the banking and financial sector. Manual evaluation can be time-consuming and prone to human bias. This project uses Machine Learning to predict whether a credit card application will be approved based on applicant information.

The application analyzes applicant details such as income, employment status, education, marital status, credit history, loan amount, and other financial attributes to provide an approval prediction. It improves decision-making while reducing processing time.

# 📊 Input Parameters

The application accepts applicant information such as:

| Parameter | Description |
|-----------|-------------|
| Gender | Applicant Gender |
| Age | Applicant Age |
| Income | Annual Income |
| Employment Status | Current Employment |
| Education | Education Level |
| Marital Status | Marital Status |
| Credit History | Previous Credit History |
| Loan Amount | Requested Credit Amount |
